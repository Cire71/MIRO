# MiroFish — Claude Code Instructions

## Commandes disponibles

### "mirofish"

Lis `mirofish.md` et exécute la procédure de démarrage complète.
Ne pose pas de questions — lance directement les services dans l'ordre indiqué.

### "campagne"

Lis `Campagne.md` et exécute la procédure de création de campagne :
pose les questions une par une, attends chaque réponse, puis génère les 4 fichiers Reality Seeds.
